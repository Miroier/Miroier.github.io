发表于 2023-02-23 阅读次数： 本文字数： 11k 阅读时长 ≈ 10 分钟

这次来学习一下更一般性的FFT分解

FFT的一般性分解

除了常见的 2 基底 FFT 以外，对于更一般的 N=N1N2N=N_1N_2N=N1​N2​ 问题，也有办法在理论上进行拆解，将问题化为数个 N1N_1N1​ 与 N2N_2N2​ 点离散傅里叶变换问题。特别的是，透过互质在数论上的特性，对于 N1N_1N1​ 与 N2N_2N2​ 互质的情况，可以进一步节省运算。

下面目前只讨论 N1N2N_1N_2N1​N2​ 非互质的情况。

为避免之后的符号混淆，我们先将 N1N2N_1N_2N1​N2​ 置换为 P1P2P_1P_2P1​P2​。

接着定义要拆分的问题，要拆分的问题为 NNN 点离散傅里叶变换，将 f[n]f\left[ n \right]f[n] 转换为 F[m]F\left[ m \right]F[m]：

F[m]=∑n=0N−1f[n]e−i2πNmn  m,n=0,1,⋯ ,N−1F\left[ m \right] =\sum_{n=0}^{N-1}{f\left[ n \right] e^{-i\frac{2\pi}{N}mn}}\,\,m,n=0,1,\cdots ,N-1 F[m]=n=0∑N−1​f[n]e−iN2π​mnm,n=0,1,⋯,N−1

直观地说，这个 NNN 点离散傅里叶变换，将由 nnn 作为参数的函数 f[n]f\left[ n \right]f[n] ，转换成由 mmm 作为参数的函数 F[m]F\left[ m \right]F[m]，并且 m,nm,nm,n 都有 NNN 个可能的数值。

待定义好要拆分的问题，便可以开始讨论如何进行拆分，基本概念是将有 NNN 个可能的数值的 m,nm,nm,n ，分别化为使用两个参数进行描述的函数 m=[m1,m2],n=[n1,n2]m=\left[ m_1, m_2 \right], n=\left[ n_1, n_2 \right]m=[m1​,m2​],n=[n1​,n2​]，并借此将原问题化为二维度离散傅里叶变换，便可使用数个较小的离散傅里叶变换问题描述整个过程。

而一种很直觉的转换方式，便是透过将 m,nm,nm,n 分别除以 P2,P1P_2,P_1P2​,P1​，以商数与余数来做为参数描述 m,nm,nm,n 的值：

n=n1P1+n2m=m1+m2P2n1,m1=0,1,⋯ ,P2−1n2,m2=0,1,⋯ ,P1−1\begin{matrix} n=n_1P_1+n_2& m=m_1+m_2P_2\\ n_1,m_1=0,1,\cdots ,P_2-1& n_2,m_2=0,1,\cdots ,P_1-1\\ \end{matrix} n=n1​P1​+n2​n1​,m1​=0,1,⋯,P2​−1​m=m1​+m2​P2​n2​,m2​=0,1,⋯,P1​−1​

其中 n1n_1n1​ 作为将 nnn 除以 P1P_1P1​ 的商数，与作为 mmm 除以 P2P_2P2​ 的余数的 m1m_1m1​ 相同，具有 P2P_2P2​ 个可能数值，同理 n2n_2n2​ 与 m2m_2m2​ 有 P1P_1P1​ 个可能数值。

将上诉的参数代换及 N=P1P2N=P_1P_2N=P1​P2​ 带入原式，可以得到：

F[m1+m2P2]=∑n1=0P2−1∑n2=0P1−1f[n1P1+n2]e−i2πP1P2(m1+m2P2)(n1P1+n2)F\left[ m_1+m_2P_2 \right] =\sum_{n_1=0}^{P_2-1}{\sum_{n_2=0}^{P_1-1}{f\left[ n_1P_1+n_2 \right] e^{-i\frac{2\pi}{P_1P_2}\left( m_1+m_2P_2 \right) \left( n_1P_1+n_2 \right)}}} F[m1​+m2​P2​]=n1​=0∑P2​−1​n2​=0∑P1​−1​f[n1​P1​+n2​]e−iP1​P2​2π​(m1​+m2​P2​)(n1​P1​+n2​)

将右边的指数部分乘开并分项化简可以得到：

F[m1+m2P2]=∑n1=0P2−1∑n2=0P1−1f[n1P1+n2]e−i2πP2m1n1e−i2πP1m2n2e−i2πP1P2m1n2e−i2πm2n1F\left[ m_1+m_2P_2 \right] =\sum_{n_1=0}^{P_2-1}{\sum_{n_2=0}^{P_1-1}{f\left[ n_1P_1+n_2 \right] e^{-i\frac{2\pi}{P_2}m_1n_1}e^{-i\frac{2\pi}{P_1}m_2n_2}e^{-i\frac{2\pi}{P_1P_2}m_1n_2}e^{-i2\pi m_2n_1}}} F[m1​+m2​P2​]=n1​=0∑P2​−1​n2​=0∑P1​−1​f[n1​P1​+n2​]e−iP2​2π​m1​n1​e−iP1​2π​m2​n2​e−iP1​P2​2π​m1​n2​e−i2πm2​n1​

最后透过 e−i2π=1e^{-i2\pi}=1e−i2π=1 与 P1P2=NP_1P_2=NP1​P2​=N，可以得到：

F[m1+m2P2]=∑n1=0P2−1∑n2=0P1−1f[n1P1+n2]e−i2πP2m1n1e−i2πP1m2n2e−i2πNm1n2F\left[ m_1+m_2P_2 \right] =\sum_{n_1=0}^{P_2-1}{\sum_{n_2=0}^{P_1-1}{f\left[ n_1P_1+n_2 \right] e^{-i\frac{2\pi}{P_2}m_1n_1}e^{-i\frac{2\pi}{P_1}m_2n_2}e^{-i\frac{2\pi}{N}m_1n_2}}} F[m1​+m2​P2​]=n1​=0∑P2​−1​n2​=0∑P1​−1​f[n1​P1​+n2​]e−iP2​2π​m1​n1​e−iP1​2π​m2​n2​e−iN2π​m1​n2​

观察上式，并加上括号辅助厘清运算顺序我们可以得到：

F[m1+m2P2]=∑n2=0P1−1{{∑n1=0P2−1f[n1P1+n2]e−i2πP2m1n1}e−i2πNm1n2}e−i2πP1m2n2F\left[ m_1+m_2P_2 \right] =\sum_{n_2=0}^{P_1-1}{\left\{ \left\{ \sum_{n_1=0}^{P_2-1}{f\left[ n_1P_1+n_2 \right] e^{-i\frac{2\pi}{P_2}m_1n_1}} \right\} e^{-i\frac{2\pi}{N}m_1n_2} \right\} e^{-i\frac{2\pi}{P_1}m_2n_2}} F[m1​+m2​P2​]=n2​=0∑P1​−1​{{n1​=0∑P2​−1​f[n1​P1​+n2​]e−iP2​2π​m1​n1​}e−iN2π​m1​n2​}e−iP1​2π​m2​n2​

最内层的运算可以视为将双参数函数 f[n1,n2]f[n_1,n_2]f[n1​,n2​] 中的一个参数 n1n_1n1​，透过离散傅里叶变换取代为由 m1m_1m1​ 描述，得到一个新函数 G1[m1,n2]G_1[m_1,n_2]G1​[m1​,n2​]（这步因为对每个不同 n2n_2n2​，都需要做一次将 n1n_1n1​ 取代为 m1m_1m1​ 的转换，共需要 P1P_1P1​ 个 P2P_2P2​ 点离散傅里叶变换）：

F[m1+m2P2]=∑n2=0P1−1{G1[m1,n2]e−i2πNm1n2}e−i2πP1m2n2F\left[ m_1+m_2P_2 \right] =\sum_{n_2=0}^{P_1-1}{\left\{ G_1\left[ m_1,n_2 \right] e^{-i\frac{2\pi}{N}m_1n_2} \right\} e^{-i\frac{2\pi}{P_1}m_2n_2}} F[m1​+m2​P2​]=n2​=0∑P1​−1​{G1​[m1​,n2​]e−iN2π​m1​n2​}e−iP1​2π​m2​n2​

下一步运算可以视为单纯的乘法，将 G1[m1,n2]G_1[m_1,n_2]G1​[m1​,n2​] 与 e−i2πNm1n2e^{-i\frac{2\pi}{N}m_1n_2}e−iN2π​m1​n2​ 相乘，得到 G2[m1,n2]G_2[m1,n2]G2​[m1,n2]（这步需要的计算量视 m1n2N\frac{m_1n_2}{N}Nm1​n2​​ 的特殊性而会有变化）：

F[m1+m2P2]=∑n2=0P1−1G2[m1,n2]e−i2πP1m2n2F\left[ m_1+m_2P_2 \right] =\sum_{n_2=0}^{P_1-1}{G_2\left[ m_1,n_2 \right] e^{-i\frac{2\pi}{P_1}m_2n_2}} F[m1​+m2​P2​]=n2​=0∑P1​−1​G2​[m1​,n2​]e−iP1​2π​m2​n2​

最后的运算可以视为将函数 G2[m1,n2]G_2[m_1,n_2]G2​[m1​,n2​] 中的 n2n_2n2​ ，透过离散傅里叶变换取代为由 m2m_2m2​ 描述，得到一个新函数 G3[m1,m2]G_3[m_1,m_2]G3​[m1​,m2​]（这步因为对每个不同 m1m_1m1​，都需要做一次将 n2n_2n2​ 取代为 m2m_2m2​ 的转换，共需要 P2P_2P2​ 个 P1P_1P1​ 点离散傅里叶变换）：

F[m(=m1+m2P2)]=G3[m1,m2]F\left[ m\left( =m_1+m_2P_2 \right) \right] =G_3\left[ m_1,m_2 \right] F[m(=m1​+m2​P2​)]=G3​[m1​,m2​]

这样就仅使用 P1P_1P1​ 与 P2P_2P2​ 点离散傅里叶变换，描述了原先的 NNN 点离散傅里叶变换。
